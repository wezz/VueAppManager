Vue App Manager Examples

This page shows different examples of how the Vue App Manager can be used.

Example 1. Render an app

<vue-app name="example1app"></vue-app>

Example 2. Render an app with attribute options

Options: { enabled: true, optionvalue:123 }
<vue-app name="example2app" options="%7B%20%22enabled%22%3A%20true%2C%20%22optionvalue%22%3A123%20%7D"></vue-app>

Example 3. Render an app with a shared store

<vue-app name="example3Aapp" store="sharedstore"></vue-app>
<vue-app name="example3Bapp" store="sharedstore"></vue-app>
